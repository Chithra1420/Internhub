# 🚀 InternHub - Smart Internship Tracker

InternHub is a web-based internship management system that helps students track internship applications, manage opportunities, and monitor application status efficiently.

## 📌 Features

### 👨‍🎓 Student Features
- User Registration & Login
- Dashboard Overview
- Browse Internship Opportunities
- Apply for Internships
- Track Application Status
- Profile Management

### 👨‍💼 Admin Features
- Admin Login
- Add Internship Opportunities
- Edit Internship Details
- Delete Internship Posts
- View Student Applications
- Manage Internship Records

## 🛠️ Technologies Used

### Frontend
- HTML5
- CSS3
- Bootstrap 5
- JavaScript

### Backend
- PHP

### Database
- MySQL
